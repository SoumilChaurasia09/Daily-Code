# 1002. Find Common Characters

**Problem Name:** Find Common Characters  
**Difficulty:** Easy

---

## Question

Given an array of strings `words`, return an array containing all characters that appear in every string, including duplicate occurrences.

The characters can be returned in any order.

---

## My Solution

```cpp
class Solution {
public:
    vector<string> commonChars(vector<string>& words) {

        vector<int> common(26, 100);

        for (int i = 0; i < words.size(); i++) {

            vector<int> freq(26, 0);

            for (char ch : words[i]) {
                freq[ch - 'a']++;
            }

            for (int j = 0; j < 26; j++) {
                common[j] = min(common[j], freq[j]);
            }
        }

        vector<string> ans;

        for (int i = 0; i < 26; i++) {

            while (common[i] > 0) {
                ans.push_back(string(1, char(i + 'a')));
                common[i]--;
            }

        }

        return ans;
    }
};
```

---

## Complexity

| Metric | Value |
|--------|-------|
| **Time Complexity** | `O(n × m)` |
| **Space Complexity** | `O(1)` *(Only fixed-size arrays of length 26 are used.)* |

Where:
- `n` = Number of words
- `m` = Average length of each word

---

## What I Learned

This problem taught me how to use **frequency arrays** to count the occurrences of characters.

Instead of comparing every character in every string, I counted the frequency of each letter in every word and kept the minimum frequency across all words. Those minimum frequencies represent the characters common to every string.

I also learned how fixed-size arrays can be very efficient when working with lowercase English letters.

---

## Personal Notes

Initially, I thought about checking each character one by one across all strings, but that quickly became complicated.

Using a frequency array of size 26 made the solution much cleaner. The key idea was realizing that the minimum frequency of each character among all words determines how many times that character should appear in the final answer.

This problem strengthened my understanding of character counting and frequency-based algorithms.
