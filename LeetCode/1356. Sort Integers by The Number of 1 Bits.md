# 1356. Sort Integers by The Number of 1 Bits

**Problem Name:** Sort Integers by The Number of 1 Bits  
**Difficulty:** Easy

---

## Question

Given an integer array `arr`, sort the integers based on the number of `1`s in their binary representation.

- If two numbers have the same number of `1` bits, sort them in ascending numerical order.

Return the sorted array.

---

## My Solution

```cpp
class Solution {
public:

    int countBits(int num) {

        int count = 0;

        while (num > 0) {
            count += num % 2;
            num /= 2;
        }

        return count;
    }

    vector<int> sortByBits(vector<int>& arr) {

        sort(arr.begin(), arr.end(), [&](int a, int b) {

            int bitsA = countBits(a);
            int bitsB = countBits(b);

            if (bitsA == bitsB) {
                return a < b;
            }

            return bitsA < bitsB;
        });

        return arr;
    }
};
```

---

## Complexity

| Metric | Value |
|--------|-------|
| **Time Complexity** | `O(n log n)` |
| **Space Complexity** | `O(1)` *(Ignoring the space used by sorting.)* |

## What I Learned

This problem helped me understand how to use a **custom comparator** with the `sort()` function.

I also learned how to count the number of `1`s in the binary representation of a number by repeatedly dividing by `2` and checking the remainder.

If two numbers have the same number of set bits, I simply compare their values and sort them in ascending order.

---

## Personal Notes

Initially, I thought I needed to convert every number into a binary string and then count the `1`s. After some practice, I realized that counting bits using division by `2` is much simpler.

This problem also introduced me to custom sorting, where I can define my own sorting rules instead of relying on the default ascending order.
