# 0448. Find All Numbers Disappeared in an Array

**Problem Name:** Find All Numbers Disappeared in an Array  
**Difficulty:** Easy

---

## Question

Given an integer array `nums` of size `n`, where each element is in the range `[1, n]`, return all the numbers in the range `[1, n]` that do not appear in the array.

---

## My Solution

```cpp
class Solution {
public:
    vector<int> findDisappearedNumbers(vector<int>& nums) {

        vector<int> freq(nums.size() + 1, 0);

        for (int i = 0; i < nums.size(); i++) {
            freq[nums[i]]++;
        }

        vector<int> ans;

        for (int i = 1; i <= nums.size(); i++) {

            if (freq[i] == 0) {
                ans.push_back(i);
            }

        }

        return ans;
    }
};
```

---

## Complexity

| Metric | Value |
|--------|-------|
| **Time Complexity** | `O(n)` |
| **Space Complexity** | `O(n)` |

---

## What I Learned

This problem helped me understand how a **frequency array** can be used to track the occurrence of numbers.

I created an array to count how many times each number appeared. After counting, I traversed the frequency array and collected the numbers whose frequency was zero. Those numbers were missing from the original array.

---

## Personal Notes

Initially, I thought about checking every number from `1` to `n` using nested loops. Although that works, it would be too slow for large inputs.

Using a frequency array made the solution much simpler and reduced the time complexity to `O(n)`. Although the follow-up asks for an `O(1)` extra space solution, this approach is easier to understand and is a good starting point before learning the optimized method.
