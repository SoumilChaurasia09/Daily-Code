# 0066. Plus One

**Problem Name:** Plus One  
**Difficulty:** Easy

---

## Question

You are given a large integer represented as an array of digits. Each element represents a single digit of the number.

Increment the integer by one and return the resulting array of digits.

---

## My Solution

```cpp
class Solution {
public:
    vector<int> plusOne(vector<int>& digits) {

        for (int i = digits.size() - 1; i >= 0; i--) {

            if (digits[i] < 9) {
                digits[i]++;
                return digits;
            }

            digits[i] = 0;
        }

        digits.insert(digits.begin(), 1);

        return digits;
    }
};
```

---

## Complexity

| Metric | Value |
|--------|-------|
| **Time Complexity** | `O(n)` |
| **Space Complexity** | `O(1)` *(Ignoring the output array)* |

---

## What I Learned

This problem taught me how to handle **carry propagation** while working with arrays.

Starting from the last digit makes the solution simple. If the current digit is less than `9`, I can increment it and return immediately. If the digit is `9`, it becomes `0`, and the carry moves to the previous digit.

I also learned how to insert a new digit at the beginning of a vector when every digit becomes `0`, such as converting `999` into `1000`.

---

## Personal Notes

At first, I thought about converting the array into an integer, adding one, and then converting it back. However, that approach wouldn't work for very large numbers because they may exceed the range of built-in data types.

Treating the array itself as a number turned out to be a much better approach. This problem was a good reminder that understanding how numbers work digit by digit can often lead to a simple and efficient solution.
