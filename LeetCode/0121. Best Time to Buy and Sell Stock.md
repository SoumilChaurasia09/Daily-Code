# 0121. Best Time to Buy and Sell Stock

**Problem Name:** Best Time to Buy and Sell Stock  
**Difficulty:** Easy

---

## Question

You are given an array `prices` where `prices[i]` represents the stock price on the `i`th day.

You can buy one stock on one day and sell it on a later day.

Return the maximum profit you can achieve. If no profit is possible, return `0`.

---

## My Solution

```cpp
class Solution {
public:
    int maxProfit(vector<int>& prices) {

        int minPrice = prices[0];
        int maxProfit = 0;

        for (int i = 1; i < prices.size(); i++) {

            if (prices[i] < minPrice) {
                minPrice = prices[i];
            }

            int profit = prices[i] - minPrice;

            if (profit > maxProfit) {
                maxProfit = profit;
            }
        }

        return maxProfit;
    }
};
```

---

## Complexity

| Metric | Value |
|--------|-------|
| **Time Complexity** | `O(n)` |
| **Space Complexity** | `O(1)` |

---

## What I Learned

This problem helped me understand how to track the minimum value while traversing an array.

Instead of checking every possible buying and selling day, I kept track of the lowest stock price seen so far. For each day, I calculated the profit if I sold the stock on that day and updated the maximum profit whenever I found a better one.

This approach solves the problem in a single traversal.

---

## Personal Notes

Initially, I thought about checking every pair of buying and selling days using two loops. Although that works, it has a time complexity of `O(n²)` and is inefficient for large inputs.

Keeping track of the minimum price while traversing the array made the solution much simpler and faster. This problem reinforced the importance of maintaining useful information during a single pass through the data.
