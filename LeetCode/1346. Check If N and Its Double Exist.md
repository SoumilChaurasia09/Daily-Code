# 1346. Check If N and Its Double Exist

**Problem Name:** Check If N and Its Double Exist  
**Difficulty:** Easy

---

## Question

Given an integer array `arr`, determine whether there exist two different indices `i` and `j` such that:

- `i != j`
- `arr[i] == 2 * arr[j]`

Return `true` if such a pair exists; otherwise, return `false`.

---

## My Solution

```cpp
class Solution {
public:
    bool checkIfExist(vector<int>& arr) {

        unordered_set<int> s;

        for (int i = 0; i < arr.size(); i++) {

            if (s.count(arr[i] * 2) || (arr[i] % 2 == 0 && s.count(arr[i] / 2))) {
                return true;
            }

            s.insert(arr[i]);
        }

        return false;
    }
};
```

---

## Complexity

| Metric | Value |
|--------|-------|
| **Time Complexity** | `O(n)` |
| **Space Complexity** | `O(n)` |

---

## What I Learned

This problem helped me understand how a **Hash Set** can be used for fast lookups.

Instead of comparing every pair of elements, I stored previously visited numbers in a hash set. For each element, I checked whether its double or half had already appeared.

This reduced the time complexity from `O(n²)` to `O(n)`.

---

## Personal Notes

My first idea was to use two nested loops to compare every pair of numbers. Although that works, it is inefficient for larger arrays.

Using a hash set made the solution much cleaner and faster. It also reinforced the importance of using the right data structure to optimize searching problems.

This problem gave me more confidence in identifying situations where hash-based lookups can replace brute-force comparisons.
