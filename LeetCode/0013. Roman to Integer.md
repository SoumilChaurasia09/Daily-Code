# 0013. Roman to Integer

**Problem Name:** Roman to Integer  
**Difficulty:** Easy

---

## Question

Roman numerals are represented using seven symbols:

| Symbol | Value |
|--------|------:|
| I | 1 |
| V | 5 |
| X | 10 |
| L | 50 |
| C | 100 |
| D | 500 |
| M | 1000 |

Normally, Roman numerals are written from largest to smallest. However, there are six special cases where a smaller numeral appears before a larger one, indicating subtraction (such as `IV = 4` and `IX = 9`).

Given a Roman numeral, convert it to an integer.

---

## My Solution

```cpp
class Solution {
public:
    int romanToInt(string s) {

        unordered_map<char, int> mp;

        mp['I'] = 1;
        mp['V'] = 5;
        mp['X'] = 10;
        mp['L'] = 50;
        mp['C'] = 100;
        mp['D'] = 500;
        mp['M'] = 1000;

        int ans = 0;

        for (int i = 0; i < s.size(); i++) {

            if (i + 1 < s.size() && mp[s[i]] < mp[s[i + 1]]) {
                ans -= mp[s[i]];
            }
            else {
                ans += mp[s[i]];
            }
        }

        return ans;
    }
};
```

---

## Complexity

| Metric | Value |
|--------|-------|
| **Time Complexity** | `O(n)` |
| **Space Complexity** | `O(1)` |

## What I Learned

This problem helped me understand how to use a **Hash Map** to store Roman numeral values.

I learned that most Roman numerals are added together, but when a smaller value appears before a larger value, it should be subtracted instead of added.

By comparing the current character with the next one, I was able to handle both normal and subtraction cases in a single traversal.

---

## Personal Notes

Initially, I thought I needed to manually handle all six special cases (`IV`, `IX`, `XL`, `XC`, `CD`, and `CM`).

Later, I realized that simply comparing the value of the current symbol with the next symbol automatically handles all these cases. This made the solution much shorter and easier to understand.

This problem also reinforced how a small observation can simplify the entire implementation.
