# 0219. Contains Duplicate II

**Problem Name:** Contains Duplicate II  
**Difficulty:** Easy

---

## Question

Given an integer array `nums` and an integer `k`, return `true` if there are two distinct indices `i` and `j` such that:

- `nums[i] == nums[j]`
- `|i - j| <= k`

Otherwise, return `false`.

---

## My Solution

```cpp
class Solution {
public:
    bool containsNearbyDuplicate(vector<int>& nums, int k) {

        unordered_map<int, int> mp;

        for (int i = 0; i < nums.size(); i++) {

            if (mp.count(nums[i])) {

                if (i - mp[nums[i]] <= k) {
                    return true;
                }

            }

            mp[nums[i]] = i;
        }

        return false;
    }
};
```

---

## Complexity

| Metric | Value |
|--------|-------|
| **Time Complexity** | `O(n)` |
| **Space Complexity** | `O(n)` |

---

## What I Learned

This problem helped me understand how a **Hash Map** can store both a value and its last seen index.

Instead of comparing every pair of elements, I stored the index of each number in a hash map. Whenever I encountered the same number again, I simply checked whether the difference between the current index and the previous index was less than or equal to `k`.

This approach reduced the time complexity from `O(n²)` to `O(n)`.

---

## Personal Notes

Initially, I thought about using two nested loops to compare every pair of elements. Although that solution works, it is too slow for large arrays.

Using a hash map made the solution much more efficient. It also helped me understand that hash maps are useful not only for storing values but also for keeping track of their positions in an array.
