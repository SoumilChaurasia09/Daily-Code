# 1528. Shuffle String

**Problem Name:** Shuffle String  
**Difficulty:** Easy

---

## Question

You are given a string `s` and an integer array `indices` of the same length.

The character at position `i` in the original string moves to position `indices[i]` in the shuffled string.

Return the shuffled string.

---

## My Solution

```cpp
class Solution {
public:
    string restoreString(string s, vector<int>& indices) {

        string ans = s;

        for (int i = 0; i < s.size(); i++) {
            ans[indices[i]] = s[i];
        }

        return ans;
    }
};
```

---

## Complexity

| Metric | Value |
|--------|-------|
| **Time Complexity** | `O(n)` |
| **Space Complexity** | `O(n)` |

---

## What I Learned

This problem helped me understand how to rearrange characters using an index mapping.

Instead of moving characters one by one inside the original string, I created a new string of the same size and placed each character directly at its correct position using the `indices` array.

This made the solution simple and efficient.

---

## Personal Notes

Initially, I thought about swapping characters repeatedly until every character reached its correct position. That approach quickly became confusing.

Using the `indices` array as a direct mapping made the implementation much easier. This problem reminded me that creating an output array or string is often the simplest way to solve rearrangement problems.
