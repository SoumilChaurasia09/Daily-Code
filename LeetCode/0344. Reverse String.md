# 0344. Reverse String

**Problem Name:** Reverse String  
**Difficulty:** Easy

---

## Question

Write a function that reverses a string. The input is given as an array of characters `s`.

The string must be reversed **in-place**, using only **O(1)** extra memory.

---

## My Solution

```cpp
class Solution {
public:
    void reverseString(vector<char>& s) {

        int left = 0;
        int right = s.size() - 1;

        while (left < right) {

            swap(s[left], s[right]);

            left++;
            right--;
        }
    }
};
```

---

## Complexity

| Metric | Value |
|--------|-------|
| **Time Complexity** | `O(n)` |
| **Space Complexity** | `O(1)` |

---

## What I Learned

This problem introduced me to the **Two Pointer** technique.

By placing one pointer at the beginning and another at the end of the string, I can swap the characters and move both pointers toward the center. This allows the string to be reversed without creating another array.

It also helped me understand how to modify an array in-place while using constant extra memory.

---

## Personal Notes

My first thought was to create another string and copy the characters in reverse order. However, that approach requires extra space and doesn't satisfy the problem's requirement.

Using two pointers made the solution much simpler and more efficient. This problem reinforced the importance of in-place algorithms and showed how the two-pointer technique can solve many array and string problems elegantly.
