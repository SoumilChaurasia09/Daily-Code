# 0268. Missing Number

**Problem Name:** Missing Number  
**Difficulty:** Easy

---

## Question

Given an array `nums` containing `n` distinct numbers in the range `[0, n]`, return the only number that is missing from the array.

---

## My Solution

```cpp
class Solution {
public:
    int missingNumber(vector<int>& nums) {

        int n = nums.size();

        int total = n * (n + 1) / 2;
        int sum = 0;

        for (int i = 0; i < n; i++) {
            sum += nums[i];
        }

        return total - sum;
    }
};
```

---

## Complexity

| Metric | Value |
|--------|-------|
| **Time Complexity** | `O(n)` |
| **Space Complexity** | `O(1)` |

---

## What I Learned

This problem taught me how mathematical formulas can simplify array problems.

Instead of checking every number individually, I calculated the expected sum of numbers from `0` to `n` using the formula `n × (n + 1) / 2`. Then I subtracted the actual sum of the array elements to find the missing number.

This approach is both efficient and uses constant extra space.

---

## Personal Notes

My first thought was to sort the array or use another data structure like a set to find the missing value. However, both approaches require extra time or memory.

Using the sum formula made the solution much simpler and more efficient. It reminded me that sometimes a mathematical observation can replace a more complicated algorithm.
