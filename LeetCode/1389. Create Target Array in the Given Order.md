# 1389. Create Target Array in the Given Order

**Problem Name:** Create Target Array in the Given Order  
**Difficulty:** Easy

---

## Question

You are given two integer arrays `nums` and `index`.

Initially, the target array is empty.

For each position `i`, insert `nums[i]` into the target array at position `index[i]`.

Return the final target array.

---

## My Solution

```cpp
class Solution {
public:
    vector<int> createTargetArray(vector<int>& nums, vector<int>& index) {

        vector<int> target;

        for (int i = 0; i < nums.size(); i++) {
            target.insert(target.begin() + index[i], nums[i]);
        }

        return target;
    }
};
```

---

## Complexity

| Metric | Value |
|--------|-------|
| **Time Complexity** | `O(n²)` |
| **Space Complexity** | `O(n)` |

---

## What I Learned

This problem helped me understand how the `insert()` function works with vectors.

I learned that a value can be inserted at any position using:

```cpp
target.insert(target.begin() + index, value);
```

Whenever a new element is inserted, the existing elements automatically shift to the right, making it easy to build the target array according to the given indices.

---

## Personal Notes

Initially, I thought I would need to manually shift elements every time I inserted a new number.

After learning about the `insert()` function, I realized that the C++ vector handles the shifting automatically. This made the solution much shorter and easier to understand.

This problem also gave me more confidence in using vector operations instead of implementing everything manually.
