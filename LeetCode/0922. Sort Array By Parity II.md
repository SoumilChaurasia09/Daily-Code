# 0922. Sort Array By Parity II

**Problem Name:** Sort Array By Parity II  
**Difficulty:** Easy

---

## Question

Given an integer array `nums` where half of the elements are even and the other half are odd, rearrange the array so that:

- Every even index contains an even number.
- Every odd index contains an odd number.

Return any valid arrangement.

---

## My Solution

```cpp
class Solution {
public:
    vector<int> sortArrayByParityII(vector<int>& nums) {

        vector<int> ans(nums.size());

        int even = 0;
        int odd = 1;

        for (int i = 0; i < nums.size(); i++) {

            if (nums[i] % 2 == 0) {
                ans[even] = nums[i];
                even += 2;
            }
            else {
                ans[odd] = nums[i];
                odd += 2;
            }
        }

        return ans;
    }
};
```

---

## Complexity

| Metric | Value |
|--------|-------|
| **Time Complexity** | `O(n)` |
| **Space Complexity** | `O(n)` |

---

## What I Learned

This problem helped me understand how to place elements directly at their correct positions.

I used two pointers:
- One pointer (`even`) for even indices (`0, 2, 4, ...`).
- Another pointer (`odd`) for odd indices (`1, 3, 5, ...`).

While traversing the array, I placed each even number at the next available even index and each odd number at the next available odd index.

---

## Personal Notes

Initially, I thought about sorting the array first and then rearranging the numbers. However, sorting isn't necessary because the problem only requires that even numbers occupy even indices and odd numbers occupy odd indices.

Using two index pointers made the solution simple, efficient, and easy to understand. It also reinforced how maintaining separate positions for different categories of elements can simplify array rearrangement problems.
