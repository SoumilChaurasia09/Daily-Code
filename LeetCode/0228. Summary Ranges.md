# 0228. Summary Ranges

**Problem Name:** Summary Ranges  
**Difficulty:** Easy

---

## Question

Given a sorted array of unique integers `nums`, return the smallest list of ranges that covers all the numbers exactly once.

- If a range contains only one number, return `"a"`.
- If a range contains multiple consecutive numbers, return `"a->b"`.

---

## My Solution

```cpp
class Solution {
public:
    vector<string> summaryRanges(vector<int>& nums) {

        vector<string> ans;

        int i = 0;

        while (i < nums.size()) {

            int start = nums[i];

            while (i + 1 < nums.size() && nums[i + 1] == nums[i] + 1) {
                i++;
            }

            int end = nums[i];

            if (start == end) {
                ans.push_back(to_string(start));
            }
            else {
                ans.push_back(to_string(start) + "->" + to_string(end));
            }

            i++;
        }

        return ans;
    }
};
```

---

## Complexity

| Metric | Value |
|--------|-------|
| **Time Complexity** | `O(n)` |
| **Space Complexity** | `O(1)` *(Ignoring the output array)* |

---

## What I Learned

This problem taught me how to identify consecutive elements in a sorted array.

I learned that by keeping track of the starting number of a range and moving forward while the numbers remain consecutive, I can easily determine whether to create a single number or a range.

It also reinforced how useful a single traversal can be when the input is already sorted.

---

## Personal Notes

Initially, I thought about checking every possible range separately, but that would make the solution more complicated.

Using a single loop and extending the range as long as consecutive numbers were found made the implementation simple and easy to understand. This problem reminded me that sorted arrays often allow efficient one-pass solutions.
