# 0389. Find the Difference

**Problem Name:** Find the Difference  
**Difficulty:** Easy

---

## Question

You are given two strings `s` and `t`.

String `t` is generated by randomly shuffling string `s` and then adding one extra character.

Return the character that was added.

---

## My Solution

```cpp
class Solution {
public:
    char findTheDifference(string s, string t) {

        int ans = 0;

        for (char ch : s) {
            ans ^= ch;
        }

        for (char ch : t) {
            ans ^= ch;
        }

        return char(ans);
    }
};
```

---

## Complexity

| Metric | Value |
|--------|-------|
| **Time Complexity** | `O(n)` |
| **Space Complexity** | `O(1)` |

---

## What I Learned

This problem introduced me to another useful application of the **XOR (`^`) operator**.

Since every character in `s` also appears in `t` except for one extra character, XORing all the characters together cancels out the matching ones. The only value left at the end is the extra character.

This approach avoids using additional data structures and solves the problem in linear time.

---

## Personal Notes

My first thought was to count the frequency of each character using a hash map. While that approach works, it requires extra space.

Using XOR made the solution much cleaner and more efficient. It also reinforced the property that `a ^ a = 0` and `a ^ 0 = a`, which is useful in many bit manipulation problems.
