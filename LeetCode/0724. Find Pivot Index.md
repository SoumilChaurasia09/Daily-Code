# 0724. Find Pivot Index

**Problem Name:** Find Pivot Index  
**Difficulty:** Easy

---

## Question

Given an integer array `nums`, find the **pivot index**.

The pivot index is the position where the sum of all elements to the left is equal to the sum of all elements to the right.

Return the leftmost pivot index. If no such index exists, return `-1`.

---

## My Solution

```cpp
class Solution {
public:
    int pivotIndex(vector<int>& nums) {

        int totalSum = 0;

        for (int i = 0; i < nums.size(); i++) {
            totalSum += nums[i];
        }

        int leftSum = 0;

        for (int i = 0; i < nums.size(); i++) {

            int rightSum = totalSum - leftSum - nums[i];

            if (leftSum == rightSum) {
                return i;
            }

            leftSum += nums[i];
        }

        return -1;
    }
};
```

---

## Complexity

| Metric | Value |
|--------|-------|
| **Time Complexity** | `O(n)` |
| **Space Complexity** | `O(1)` |

---

## What I Learned

This problem introduced me to the concept of **Prefix Sum**.

Instead of calculating the left and right sums separately for every index, I first found the total sum of the array. Then, while traversing the array, I maintained the left sum and calculated the right sum using:

```text
rightSum = totalSum - leftSum - nums[i]
```

This allowed me to find the pivot index in a single traversal.

---

## Personal Notes

Initially, I thought about using two nested loops to calculate the left and right sums for every index. However, that approach would take `O(n²)` time.

Using the total sum and updating the left sum as I traversed the array made the solution much more efficient. This problem helped me understand how prefix sum techniques can eliminate unnecessary repeated calculations.
