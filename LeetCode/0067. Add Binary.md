# 0067. Add Binary

**Problem Name:** Add Binary  
**Difficulty:** Easy

---

## Question

Given two binary strings `a` and `b`, return their sum as a binary string.

---

## My Solution

```cpp
class Solution {
public:
    string addBinary(string a, string b) {

        string ans = "";

        int i = a.size() - 1;
        int j = b.size() - 1;
        int carry = 0;

        while (i >= 0 || j >= 0 || carry) {

            int sum = carry;

            if (i >= 0) {
                sum += a[i] - '0';
                i--;
            }

            if (j >= 0) {
                sum += b[j] - '0';
                j--;
            }

            ans = char(sum % 2 + '0') + ans;

            carry = sum / 2;
        }

        return ans;
    }
};
```

---

## Complexity

| Metric | Value |
|--------|-------|
| **Time Complexity** | `O(max(n, m))` |
| **Space Complexity** | `O(max(n, m))` |

## What I Learned

This problem helped me understand how binary addition works using the same idea as normal addition.

I learned to start from the last character of both strings, add the digits along with the carry, and build the answer one bit at a time. It also showed me how characters can be converted into integers using `'0'`.

---

## Personal Notes

Initially, I thought about converting the binary strings into decimal numbers, adding them, and converting the result back to binary. However, that approach wouldn't work for very large binary strings because they can exceed the range of built-in integer types.

Simulating the addition process directly was much simpler and worked for all valid input sizes. This problem also gave me more confidence in handling strings and carry-based calculations.
