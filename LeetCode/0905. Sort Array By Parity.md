# 0905. Sort Array By Parity

**Problem Name:** Sort Array By Parity  
**Difficulty:** Easy

---

## Question

Given an integer array `nums`, move all the even integers to the beginning of the array, followed by all the odd integers.

Return any array that satisfies this condition.

---

## My Solution

```cpp
class Solution {
public:
    vector<int> sortArrayByParity(vector<int>& nums) {

        int left = 0;
        int right = nums.size() - 1;

        while (left < right) {

            if (nums[left] % 2 > nums[right] % 2) {
                swap(nums[left], nums[right]);
            }

            if (nums[left] % 2 == 0) {
                left++;
            }

            if (nums[right] % 2 == 1) {
                right--;
            }
        }

        return nums;
    }
};
```

---

## Complexity

| Metric | Value |
|--------|-------|
| **Time Complexity** | `O(n)` |
| **Space Complexity** | `O(1)` |

---

## What I Learned

This problem helped me understand another application of the **Two Pointer** technique.

I placed one pointer at the beginning of the array and another at the end. If an odd number was found on the left side and an even number on the right side, I swapped them. By moving the pointers inward, all even numbers naturally moved to the front while odd numbers shifted to the back.

This approach solves the problem in-place without using any extra memory.

---

## Personal Notes

At first, I thought about creating two separate arrays—one for even numbers and one for odd numbers—and then combining them. Although that approach works, it requires additional space.

Using two pointers made the solution much more efficient and easier to understand. This problem reinforced how useful the two-pointer technique is for rearranging elements within an array.
