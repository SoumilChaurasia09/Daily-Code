# 0217. Contains Duplicate

**Problem Name:** Contains Duplicate  
**Difficulty:** Easy

---

## Question

Given an integer array `nums`, return `true` if any value appears at least twice in the array. Otherwise, return `false`.

---

## My Solution

```cpp
class Solution {
public:
    bool containsDuplicate(vector<int>& nums) {

        unordered_set<int> s;

        for (int i = 0; i < nums.size(); i++) {

            if (s.count(nums[i])) {
                return true;
            }

            s.insert(nums[i]);
        }

        return false;
    }
};
```

---

## Complexity

| Metric | Value |
|--------|-------|
| **Time Complexity** | `O(n)` |
| **Space Complexity** | `O(n)` |

---

## What I Learned

This problem helped me understand how an **unordered_set** can quickly detect duplicate elements.

Instead of comparing every pair of elements, I stored each number in a hash set. Before inserting a number, I checked whether it already existed in the set. If it did, I immediately returned `true`.

Using a hash set reduced the time complexity from `O(n²)` to `O(n)`.

---

## Personal Notes

My first idea was to use two nested loops and compare every element with the others. Although that approach works, it becomes very slow for large arrays.

Using an `unordered_set` made the solution much simpler and more efficient. This problem reinforced the importance of choosing the right data structure when checking for duplicates.
